# 쿼리 컨벤션

## 문서 목적

이 문서는 SQL 쿼리의 컨벤션 준수 여부를 비교하고 판정하기 위한 규칙 정의서이다.
각 규칙은 `규칙 ID`, `적용 범위`, `판정 기준`을 기준으로 해석하며, 필요한 경우 `위반 예시`, `준수 예시`를 보조 기준으로 사용한다.

## 공통 판정 원칙

- 예시와 동일하지 않은 쿼리라도 `판정 기준`의 의미를 만족하면 준수로 판정한다.
- 예시와 동일하지 않은 쿼리라도 `판정 기준`을 어기면 위반으로 판정한다.

## QUERY-SELECT-001: SELECT * 사용을 지양한다

### 적용 범위

- SQL `SELECT` 문
- MyBatis XML Mapper 또는 어노테이션 기반 쿼리

### 판정 기준

- `SELECT * FROM` 사용을 지양한다.
- 조회에 필요한 필드만 명시적으로 `SELECT` 절에 작성한다.
- 모든 컬럼이 필요하더라도 실제로 사용하는 컬럼 목록을 명시한다.

### 위반 예시

```sql
SELECT *
FROM users
```

### 준수 예시

```sql
SELECT
    user_id,
    user_name,
    email
FROM users
```
